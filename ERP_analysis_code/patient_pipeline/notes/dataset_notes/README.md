# Auditory Aphasia dataset analysis

Marker analysis of the auditory aphasia dataset + required data cleaning

## Contents
- `marker_overview.md`: Overview of all unique markers for every patient.
- `scripts/`: Python scripts to generate the .log files with marker information. 
    - `marker_analysis.py`: Main script to run the analysis
    - `marker_functions.py`: Functions used in the analysis
- `markers/`: Results generated by the script
    - `contents.txt` Explanation of the contents of the two subfolders below.
    - `odd_markers/`
    - `unique_markers/`
- `data_cleanup.md`: All needed data cleaning after analyzing the contents of `markers/`.
- `session_conditions.md`: Overview of all conditions (6D/HP and SOA in ms) per patient.

## How to run

1. Open `marker_analysis.py`
2. At the top of the script, fill in the following:
```
data_path = "path/to/patient" 
patient_number = ...
last_session_number = ...
```
3. Make sure that the .vhdr files of each session are stored in ``` data_path + "/P1_S1/anonymized" ``` for patient 1, session 1, for instance.
