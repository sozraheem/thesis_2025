# Auditory Aphasia dataset notes

This README contains instructions on how to navigate through the contents of this folder.

## Contents
- `marker_overview.md` contains an overview of all unique markers for every patient.
- The folder `scripts` contains the Python script to generate the .log files with marker information. 
- I have saved the output of the script in the folder `markers/`. This folder contains two subfolders and `contents.txt`, explaining the contents of the subfolders.
- In `data_cleanup.md` I have documented every change I intended to make after analyzing the contents of `markers/`.
- In `session_conditions.md` all conditions (6D/HP and SOA in ms) are listed per patient.
